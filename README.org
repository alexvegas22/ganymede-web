#+title: Application Web Ganymède
#+author: Alexandre Baudouin, Julien Giguère, Mohamed Fateme, Andrew Tremblay
#+date: 2024-01-12

Our goal with this project is to provide a platform for volunteers and donors to help the homeless living in Côte-Des-Neiges, letting people sign up as volunteers, giving them schedules and itineraries. The donations page informs on how to help the initiative.

Installation:

Nécéssite .NET 8.0 et peut être éxécuté à l'aide de visual studio ou VS code.
https://learn.microsoft.com/en-us/dotnet/core/tutorials/with-visual-studio?pivots=dotnet-8-0
